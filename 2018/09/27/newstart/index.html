windows在官网下载安装包之后解压，运行nginx.exe即可运行服务器

下面是一些简单的指令：
1、启动：

C:\server\nginx-1.0.2>start nginx

或

C:\server\nginx-1.0.2>nginx.exe

注：建议使用第一种，第二种会使你的cmd窗口一直处于执行中，不能进行其他命令操作。

2、停止：

C:\server\nginx-1.0.2>nginx.exe -s stop

或

C:\server\nginx-1.0.2>nginx.exe -s quit

注：stop是快速停止nginx，可能并不保存相关信息；quit是完整有序的停止nginx，并保存相关信息。

3、重新载入Nginx：

C:\server\nginx-1.0.2>nginx.exe -s reload

当配置信息修改，需要重新载入这些配置时使用此命令。

4、重新打开日志文件：

C:\server\nginx-1.0.2>nginx.exe -s reopen

5、查看Nginx版本：

C:\server\nginx-1.0.2>nginx -v

nginx做反向代理服务器

1、更改nginx的配置文件

更改nginx.conf文件，在http/server/location层次结构下，添加proxy_pass http://localhost:7080; 一句话即可。注意不要放了;结尾。

server {
        listen       80;
        server_name  localhost;

        #charset koi8-r;

        #access_log  logs/host.access.log  main;

        location / {
            root   html;
            index  index.html index.htm;
             proxy_pass  127.0.0.1:8080
       }

3、重启nginx，在浏览器中输入nginx的访问地址，显示的就是Tomcat的访问地址。

4、现在虽然nginx反向代理成功了，但是为了更好的发挥nginx的性能。我们将Tomcat的js、html、图片等静态文件配置到nginx上进行缓存，这样就可以提高应用的访问效率了。

5、删除 location / 节点，避免全部请求被拦截。新增 location ~ .jsp$ 节点和 location ~ .(html|js|css|png|gif|jpg)$ 节点，如下:

server {
    listen       80;
    server_name  localhost;

    #charset koi8-r;

    #access_log  logs/host.access.log  main;


    location ~ \.jsp$ {
       proxy_pass  127.0.0.1:8080
    }

    location ~ \.(html|js|css|jpg|png|gif)${
         root tomcat
}

6、访问http://127.0.0.1，返回nginx 404。这是因为匹配的location，应用自动跳转到nginx的404.

访问http://127.0.0.1/index.jsp，返回到tomcat页面。因为jsp会自动匹配配置的location。

访问http://127.0.0.1/tomcat.png，因为tomcat.png放在根目录下的tomcat文件里里，也能正确显示这个logo。说明nginx配置的静态文件location是生效的。

至此，Nginx配置反向代理Tomcat完成。

负载均衡配置

假设有两台tomcat服务器，在同一台机器上做测试时可以开两个tomcat进程，前提是两个tomcat中的所有端口都要不同

upstream mytomcats {  

      server 127.0.0.1:8080;  

      server 127.0.0.1:8081;  

    }  

    server {
        listen       80;
        server_name  localhost;

        #charset koi8-r;

        #access_log  logs/host.access.log  main;


        location ~ \.jsp$ {
           proxy_pass  http://mytomcats;
        }

        location ~ \.(html|js|css|jpg|png|gif)$ {
             root tomcat;
        }
